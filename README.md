## 设计宗旨

- 重视四人本团本以及非高难本的体验，不强调高难本（期望蓝紫）或打得高，重视随机应变。
- 所有判断主要以自身或敌人状态（buff）、技能冷却时间（cooldown）、职业量谱（gauge）来判断接下来要用什么技能。尽可能少用之前技能的记录、自定义字段等。
