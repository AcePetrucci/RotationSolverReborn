# Rotation Settings

In `Rotation` Panel, you'll see a lot of jobs to choose.

![Rotation Settings](assets/image-20230417211307606.png)

The jobs are grouped by role in individual panel, which contains `Tank`, `Melee`, `Healer`, `Ranged`, `Magical` and `Info`.

## Jobs

On the first four panels, there are some settings about the job.

![Jobs Config](assets/1681737493526.png)

`Enable` is a toggle to choose whether using this rotation during combat.

`Job` is a text to show the job. If its color is Yellow, which means you are this job right now.

`Name` shows the rotation name of the rotation. You can choose another rotation if you want.

![Rotation changing](assets/1681737718208.png)

`Assembly` shows the assembly this rotation belongs. The color of this contains a lot of information of rotation.  Orange means `Beta`, purple means `Third-party`, Red means `Invalid`.

![Beta](assets/image-20230417213617411.png)

![Third Party](assets/image-20230417213701457.png)

![Invalid](assets/image-20230417213730880.png)

`Game version` shows the game version when this rotation was written.

`Reset Config` can reset the config of the `Job Config` not `Role Config`.

`Source Code` can open a web browser by url is the author wrote one.

`Desc` is a picture or a link to the description about this rotation.

`Job Config` is a set of config that the author of this rotation wrote. You can use the macro if your job is the same to this rotation's job.

### Job Config

Job config is a big thing about the config. Different role gets different configs.

`Hostile target filtering condition` is a general option for every role. It controls which targets should be regarded as hostile targets. It contains three options. Choose it if you need.

`HP threshold for tank to use invincibility` is a trigger of using invincibility. If the percent of HP of your character is lower than what you set here, it'll automatically use the invincibility for you.

## Info

`Info` contains all assemblies that you loaded into this plugin. It shows the `Author`, `Version`, `Name`, `Rotations` and `Links` of this assembly.

![Rotation Info](assets/image-20230417213203296.png)

You can click on `Name` button to show the file of this assembly in your computer.

Purple means Third-party assembly.

`Donwload Rotations` is a force downloading for rotations. If your rotations didn't work well, please try to click on it.

`Download Rotations` is an option for whether downloading rotations, if uncheck it, it'll not download any rotations for you. In the plugin, it contains nothing about rotations, so please check on it.

`Auto Update Rotations` is an option for auto update the rotations from url for you.
